# Routine Execution

Use this skill for heartbeats, scheduled routines, and recurring company checks.

## Rules

- Work through the assigned Paperclip issue or routine execution.
- Use the routine objective exactly as written.
- Keep the run bounded and observable.
- Record command output or evidence summaries.
- End with a formal disposition.
- Do not mutate production state unless the routine explicitly includes an
  approved action.

If the routine cannot run, mark it blocked with the exact missing input.
