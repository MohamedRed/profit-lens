# Vehicle Preset Baselines (France)

ProfitLens ships with editable baseline values for vehicle maintenance and
depreciation to reduce the amount of data a deliverer must enter.

These values are starting points only and should be reviewed by each user.
They are stored per vehicle and can be overridden at any time in the app.

## Baseline values (EUR/km)

| Vehicle type | Maintenance | Depreciation |
| --- | --- | --- |
| Bike | 0.02 | 0.01 |
| E-bike | 0.03 | 0.05 |
| Scooter | 0.04 | 0.06 |
| Car | 0.05 | 0.12 |

Consumption defaults are handled separately and, for cars, refined by the
ADEME car labelling dataset.
